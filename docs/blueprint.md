# **App Name**: Checking Central

## Core Features:

- PI Data Fetch: Retrieve campaign data based on PI number.
- Form Generation: Dynamically generate file upload fields based on media type.
- Client-Side Validation: Perform input and file validation before submission.
- Progress Updates: Display real-time progress updates during file upload.
- Data Submission to Central Server: Submit the form data, including files, to a central server using XMLHttpRequest.

## Style Guidelines:

- Primary color: Deep blue (#0033A0), echoing the Grupo OM brand, symbolizing trust and professionalism.
- Background color: Light gray (#F7FAFC), offering a clean and neutral backdrop to focus attention on the form.
- Accent color: Vibrant orange (#F59E0B), drawn from the brand's gradient, highlighting interactive elements and important notices.
- Body and headline font: 'Inter' sans-serif for a modern and easily readable style across all form elements and headings.
- Note: currently only Google Fonts are supported.
- Use clear and functional icons from a set like Heroicons to represent form fields and actions.
- Maintain a clean and organized layout using a grid system to ensure readability and ease of navigation.
- Employ subtle animations for form interactions, such as progress bar updates and success/error messages.
